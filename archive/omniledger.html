OmniLedger Login Service

Here you can find a walkthrough to explore what the OmniLedger login can do. Feel free to send questions to the C4DT Factory.

Register User

As a first step, you need to register your user on the blockchain. The docker images for this demo contain already an initialized blockchain, so you can simply use the following link to

By signing up to the local OmniLedger blockchain, a secret key is stored in your browser. The new state of the blockchain is also stored in the node/* directories, with one directory for each node. Once you registered, you're ready to go to the next step.

Update User Attributes

Now that you're signed up, you can change your name and see the new block on the chain: go to Contact Information, change your alias and then click on Update Credential. You'll see a small animation of a block being included in the blockchain, and your credential is updated.

Simple block explorer

The user interface also contains a very simple block explorer at the bottom. Click on the block Latest blocks and then on the block to the right. You can see the creation date of the block, and links to the previous blocks. In the lower part, click on ByzCoin Body, and you'll the instruction in the block representing your change of name. One of the standalone features of ByzCoin are the multilevel links, which you can see as blue rectangles in the BackwardLinks and ForwardLinks fields. This allows for very fast catchup mechanisms and very thin clients.

Creating a new User

Let's create a second credential and move tokens between the two. Switch to the Contacts Tab and click on Create Contact. Give it an alias and choose Admin for the Default view, then click on Create. Once the animation with the block adding finishes, open the New User Signup link in a Private Navigation Tab. It is important not to open the link in the main browser, else it will overwrite your credential, and you will have no possibility to recover it - besides starting from scratch as described in the main document. Confirm by clicking on Register User and Start Demonstrator.

Now you have two credentials open: your original credential, and the one in the private navigation. In your main credential, click on the Transfer link next to the name of the new credential you just created. Enter a number between 1 and 100 to transfer some tokens to the new credential. When confirming with Transfer, once the animation is done, you see the Memory field of the new credential updating with the number of tokens you transferred. You can also look at this transaction in the blockchain explorer at the bottom of the page.
